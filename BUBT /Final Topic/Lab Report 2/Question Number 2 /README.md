Write a Java program that reads a line of text from a file named input.txt. The program should process the line by splitting it into individual words, converting each word to uppercase, and replacing all vowels with the character ‘#’. After processing, the modified words should be combined into a single line and written into a new file named output.txt.

## Sample input.txt content:

java programming lab

## Sample output.txt content:

J#V# PR#GR#MM#NG L#B
