Write a Java program that demonstrates multithreading by creating two separate threads. One
thread should print all odd numbers from 1 to 10, while the other thread should print all even
numbers from 1 to 10. Both threads should run concurrently and display their outputs independently. The program should be implemented using either the Thread class or the Runnable interface.

## Sample Output:
Odd Thread: 1 3 5 7 9

Even Thread: 2 4 6 8 10
