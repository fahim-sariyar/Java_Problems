Write a Java program that demonstrates the use of an abstract class and an interface. Create an abstract class named Shape with an abstract method to calculate area. Also create an interface named Display with a method to show output. Then implement two classes, Circle and Rectangle, which extend the Shape class and implement the Display interface. In the main method, create objects of these classes, calculate their areas, and display the results.

## Sample Output:
Circle Area: 78.5
Rectangle Area: 50.0
