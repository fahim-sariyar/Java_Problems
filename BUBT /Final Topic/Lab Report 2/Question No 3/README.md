Write a Java program that takes marks of five students as input from the user and performs appropriate exception handling. The program should handle cases where the input is not an integer and also ensure that the marks are within the range of 0 to 100. If a mark is entered as -1, it should be treated as an absent case and considered as zero. After processing all valid inputs, the program should calculate and display the average marks using proper try-catch-finally blocks.

## Sample Input:

Marks: 80 75 -1 90 110

## Sample Output:
Invalid mark detected
Average Marks: 61.25
