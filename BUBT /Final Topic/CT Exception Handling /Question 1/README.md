# A library stores book titles in an array.

While accessing books, sometimes an invalid index may be used, or a book entry may be missing. The system should handle these cases and continue execution.

Write a Java program using nested try statements and proper exception handling to display meaningful messages for such cases.

## Expected Output:

Accessing index 1:
Book: Database System

Accessing index 2:
Book entry is missing.

Accessing index 3:
Invalid index.
