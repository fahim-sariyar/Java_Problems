In a company, employee salary is calculated based on working hours. The system should not accept negative hours or hours greater than 10. If such input is given, the program should handle it properly and continue execution.

Write a Java program that uses throw, throws, and exception propagation to handle invalid working hours.

## Expected Output:

Working hours: 8
Salary: 4000.0

Working hours: -3
Exception caught: Working hours cannot be negative.

Working hours: 30
Exception caught: Working hours cannot exceed 10.
