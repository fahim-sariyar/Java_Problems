# You are developing a file upload system where users can upload files based on size. 

The system should not allow uploading files smaller than 1 MB or larger than 100 MB. If such a case occurs, the system should safely handle it without crashing. Write a Java program that uses a user-defined exception along with throw, throws, try, and catch to handle invalid file sizes in the main method.

### Expected Output: 
Uploading file of size: 50

Upload successful!

Uploading file of size: 150

Exception caught: Invalid file size.
