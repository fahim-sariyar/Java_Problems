In a weather monitoring system, temperature readings (in degrees Celsius) are collected
from different sensors. Due to repeated data transmission, some temperature values may
be recorded multiple times. Write a Java program that uses a Set collection to store
temperature values. Create a class named Temperature with one attribute (value) and an
appropriate constructor. Ensure duplicate values are automatically removed. Insert at
least 6 Temperature objects including duplicate values in the main method. Display the
stored temperatures in descending order.
