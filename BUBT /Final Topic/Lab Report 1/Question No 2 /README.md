Write a Java program that defines a class named Student containing attributes for name, id, and marks. The class should include a method that calculates and returns the grade of the student based on the marks, where marks 80 and above correspond to grade A, marks between 60 and 79 correspond to grade B, marks between 40 and 59 correspond to grade C, and marks below 40 correspond to grade F. In the main method, create an object of the Student class, assign values to the attributes, and display the student’s information along with the calculated grade.

## Sample Input:
Name: Rahim
ID: 123
Marks: 75

## Sample Output:
Name: Rahim
ID: 123
Marks: 75
Grade: B
