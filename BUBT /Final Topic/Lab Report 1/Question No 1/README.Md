Write a Java program that takes an integer value N from the user. The program should determine
whether the number is positive, negative, or zero. If the number is positive, it should calculate the
sum of all integers from 1 to N and also display all even numbers within that range using a loop.
If the number is negative, the program should display an appropriate message instead of
performing the calculations.

## Sample Input:
Enter a number: 6

## Sample Output:
The number is Positive
Sum from 1 to 6 is: 21
Even numbers: 2 4 6
