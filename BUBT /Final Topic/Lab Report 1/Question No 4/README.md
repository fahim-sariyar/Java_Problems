Write a Java program that demonstrates inheritance and runtime polymorphism using a base class named Employee and two derived classes named Manager and Developer. The Employee class should contain a method to calculate salary, which will be overridden in the derived classes. The Manager class should calculate salary by adding a bonus with the base salary, while the Developer class should calculate salary by adding overtime with the base salary. In the main method, use runtime polymorphism to call the appropriate method and display the salary for each type of employee.

## Sample Output:
Manager Salary: 60000
Developer Salary: 45000
