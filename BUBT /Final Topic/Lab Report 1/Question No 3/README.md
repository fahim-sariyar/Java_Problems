Write a Java program that defines a class named Calculator to demonstrate the use of constructors and method overloading. The class should include both a default constructor and a parameterized constructor. It should also contain overloaded methods to perform addition of two integers, addition of three integers, and addition of two double values. In the main method, create objects and display the results of all addition operations.

## Sample Output:
Addition of two integers: 10

Addition of three integers: 15

Addition of two doubles: 10.5
